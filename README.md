# Portfolio & Blog

A modern, responsive portfolio website showcasing projects, blog posts, case studies, and creative explorations.

## Features

- Clean, modern design with Tailwind CSS
- Responsive navigation with sticky header
- Hero section with call-to-action buttons
- Content sections for Blog, Projects, Case Studies, and Explorations
- Recent posts preview with article cards
- Newsletter signup and footer with social links

## Tech Stack

- **Vite** - Build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **PostCSS** - CSS processing with Autoprefixer
- **Puppeteer** - Web scraping and automation

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
