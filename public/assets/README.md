# Assets Directory

This directory contains static assets for the website.

## CV/Resume

Place your CV PDF file here with the filename:

- `nolan-figueroa-cv.pdf`

The CV button in the bottom drawer will link to this file at `/assets/nolan-figueroa-cv.pdf`

## File Organization

- Keep filenames lowercase with hyphens
- Use descriptive names
- Organize by type if the directory grows (images/, documents/, etc.)
