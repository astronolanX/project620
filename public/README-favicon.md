# Favicon Files

This directory contains favicon files for the website.

## Current Files

- `favicon.svg` - Modern SVG favicon (recommended, scalable)
- `favicon.ico` - Traditional ICO format for older browsers

## Replacing the Favicon

To replace the placeholder favicon with your own:

1. **For SVG favicon**: Replace `favicon.svg` with your custom SVG design
2. **For ICO favicon**: Replace `favicon.ico` with your custom ICO file

## Favicon Specifications

### SVG Favicon

- Size: 32x32 viewBox (scalable)
- Format: SVG
- Should work well at small sizes
- Modern browsers support this format

### ICO Favicon

- Recommended sizes: 16x16, 32x32 (multi-size ICO)
- Format: ICO
- Fallback for older browsers

## Design Tips

- Keep it simple - favicons are displayed very small
- Use high contrast
- Consider how it looks on both light and dark browser themes
- Test at actual favicon size (16x16 pixels)
